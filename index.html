About me

I'm Aditya Mohan - a Computer Science graduate from BITS Pilani, Goa. I'm currently interested in the domain of large-scale data management systems and I love to play around with tasks relating to big data storage, management and processing. I firmly believe in the importance of such scalable systems given the "data explosion" and the need to obtain valuable information from disorganized data. Apart from my research interests, I enjoy taking up the occassional side projects in multiple areas. Outside of work I enjoy spending time playing instruments (particularly the guitar), quizzing, and sports to mention a few.

Work

Consistent Multi- and Single-view HDR Image Reconstuctions from Single Exposures using Deep Learning Techniques

Worked with Dr. Celine Loscos and Dr. Remi Cozot on reconstructing multi-view HDR images from single exposure LDR images using a Deep Learning algorithm while achieving consistency among multiple views. Short paper under review for Eurographics '22.

Enhancing a Performance Prediction Tool for CUDA kernels

Worked under the supervision of Dr. Santonu Sarkar. The primary objective of this project is to build a performance prediction Java based tool using a combination of static and dynamic analysis of the CUDA Program. The execution time of the CUDA Program is to be measured and compared with the current models.

Computational Photography Techniques

Creating an online repository for various image processing and manipulation techniques applied on some photos captured by myself. Looking forward to expanding the repository to incorporate more techniques in Computational Photography.

Hyperspectral Image Segmentation

This project was carried out as part of the "Pixxel Open Data challenge". The task was to segment hyperspectral satellite imagery using an unsupervised classification technique. The 242 bands of satellite imagery data was collected by NASA's EO1 program.

Automatic Leather defect classification

This project was part of my internship. Apart from a lot of literature survey on employing ML techniques, I came up with a simple ML classifier which makes use of the Random Forest Classifier to classify leather samples into defective and non-defective samples.
