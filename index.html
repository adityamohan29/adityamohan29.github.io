  • Enhancing a Performance Prediction Tool for CUDA kernels

    BITS Pilani Goa

    ( Jan 2020 - Present )

    Working under the supervision of Dr. Santonu Sarkar and Ms. Gargi Alavani. The primary objective of this project is to build a performance prediction Java based tool using a combination of static and dynamic analysis of the CUDA Program. The execution time of the CUDA Program is to be measured and compared with the current models. Currently, I'm attempting on modifying the previous model to decrease prediction error by running various benchmarks on the GPU. The GPU used was NVIDIA Tesla K20, however, I'll continue working on extending the tool to other architectures (like Pascal) as well.

  • Hyperspectral Image Segmentation

    ( May 2020 )

    This project was done as part of the "Pixxel Open Data challenge". The task was to segment hyperspectral satellite imagery using an unsupervised classification technique. The 242 bands of satellite imagery data was collected by NASA's EO1 program. There were three main aspects to this task :-

  • 1. Data Importing and Preprocessing - Done primarily using the tifffile and rasterio libraries
  • 2. Dimensionality Reduction - Incremental Principal Component Analysis was employed to reduce the band dimensions and for further analysis.
  • 3. Segmentation - The three primary principal components were segmented individually and stacked together to form a false color RGB image.
